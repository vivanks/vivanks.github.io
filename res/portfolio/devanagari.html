Brain can easily able to recognise handwritten words which computers are still struggling to figure out what they are. This amazing power of brain requires to be transferred to computer for automatic digitalisation of handwritten texts and manuscript. The work I proposed in this project tried to make a model which is able to recognise and predict different Hindi character which can be further used to identify and digitalise the whole page or text. Model uses different approaches like base-line neural network , convolutional neural network and a simple logistic regression model.
